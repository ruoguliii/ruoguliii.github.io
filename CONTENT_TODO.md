# Personal Website Content Checklist

The imported website currently keeps the template author's content as visual
placeholders. Replace items progressively; no section needs to be completed all
at once.

## Identity

- English name
- Chinese name
- Current title and affiliation
- Short biography
- Portrait
- Page title, description, and favicon

## Contact and profiles

- Email
- CV
- GitHub
- Google Scholar
- WeChat QR code (optional)

## Main sections

- News
- Research interests
- Publications and project links
- Honors and awards
- Education and professional experience

## Media

- Publication thumbnails
- Institution logos
- Award images
- Project videos, models, and PDFs

## Final publishing checks

- Replace every remaining reference to the template author
- Verify all external links
- Review mobile layout
- Add the correct analytics or site-verification settings, if wanted
- Update the footer and last-updated date
